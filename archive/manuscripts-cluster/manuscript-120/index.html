Name

Antigone C.R.

Narrative:Apocalyptic

Object:Has a Motor

Domain:Health

[HAND CRANK USED TO ACTIVATE VACUM. MOTOR COMPARTMENT - VENT - Vacuum] A machine that cleans and disinfects clothing that has zombie blood on it. The blood turns into vapor and is let out through vents in the back. It is small and compact so that travel with the object is possible. In the case of a non-zombie apocalyptic world, the object can be used to lift dirt from the body. The person using the object would have to be covered in water so that the device that works. Inside the object is a motor that runs on liquid.
